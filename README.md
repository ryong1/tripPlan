# ✈️ 여행 플래너

친구들과 **실시간으로 함께** 여행 계획을 세우는 웹앱입니다.
한 명이 편집하면 접속한 모두에게 바로 반영됩니다.

## 기능
- 🧳 **여행 생성** — 목적지 + 기간(출발~도착)을 넣으면 날짜별 일정이 자동 생성
- 📅 **일정표(타임테이블)** — 장소를 **지도 검색**해서 추가, 시간 지정, 항목 클릭 시 펼쳐서 상세 입력
- 🧭 **동선 계산** — OSRM 실제 경로로 구간별 이동거리·시간 표시, `동선 최적화`로 최단 방문순서 자동 정렬
- 📍 **가고싶은 곳 + 투표** — 검색으로 후보를 모으고 👍 투표로 결정, 정한 곳을 일정에 바로 담기
- 💰 **경비 정산** — 지출 기록 → 1인당 정산 + "누가 누구에게 얼마" 송금 방법까지 자동 계산
- 🎒 **준비물 체크리스트** — 담당자 지정, 완료 체크

## 지도 / 경로
- 장소 검색: **OpenStreetMap Nominatim** (서버가 프록시, API 키 불필요)
- 동선 계산: **OSRM**(자동차 기준 실제 경로) — 공용 서버라 가끔 지연/실패할 수 있어요
- 좌표가 있는 항목(📍)만 동선 계산에 포함됩니다
- 장소를 이름만 입력해도 서버가 위치를 자동 조회(지오코딩)해 이동시간 계산에 포함시켜요

## 대중교통 길찾기 (선택 기능)
일정표의 각 날짜에서 `🚗 자동차 / 🚆 대중교통`을 토글하면 구간별 이동수단을 바꿔볼 수 있어요.
대중교통(지하철·버스·기차)은 **ODsay 무료 API 키**가 있어야 동작합니다.
1. https://lab.odsay.com 에서 무료 회원가입 후 API 키 발급
2. 발급받은 키를 아래 둘 중 하나로 설정 (**재시작 없이 적용**)
   - `data/odsay.key` 파일을 만들고 키만 붙여넣기, 또는
   - 환경변수 `ODSAY_API_KEY` 로 설정
3. 키가 없으면 대중교통 토글 시 "🔑 ODsay 키 설정 필요"로 안내되고, 자동차 계산은 그대로 사용돼요
> 무료 티어는 일일 호출 제한이 있어, 같은 구간 결과는 캐싱해서 재호출을 줄입니다.

## 실행 방법
```bash
npm install      # 최초 1회
npm start        # 서버 실행
```
브라우저에서 http://localhost:3000 접속

## 함께 쓰는 법
1. 이름을 입력하고 **여행 만들기**
2. 우측 상단 **🔗 공유** 버튼 → 링크 복사 → 카톡 등으로 전달
3. 링크를 연 사람은 이름만 입력하면 바로 같이 편집

> 같은 Wi-Fi(집/회사)에 있는 사람과 쓰려면 링크의 `localhost`를
> 내 PC의 IP(예: `192.168.0.10`)로 바꿔 공유하세요.
> 외부(인터넷)에서 접속하려면 배포가 필요합니다 — 원하면 도와드릴게요.

## 데이터
- 모든 여행 데이터는 `data/trips.json` 파일에 저장됩니다.
- 서버를 껐다 켜도 계획은 유지됩니다.
